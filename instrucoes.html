Odonto_FBBR - Versão Portátil

Bem-vindo à versão portátil do Odonto_FBBR! Este pacote foi criado para permitir que você acesse todo o conteúdo do site diretamente do seu pendrive, sem necessidade de instalação.

Como usar:

  1. Copie toda a pasta Odonto_FBBR para seu pendrive
  2. Para iniciar o aplicativo, dê um duplo clique no arquivo iniciar_odonto_fbbr.bat
  3. O site será aberto automaticamente no seu navegador padrão
  4. Navegue pelo conteúdo normalmente, como faria no site online
Nota: Todos os recursos funcionam offline, incluindo o assistente de IA e o sistema de filtros. Não é necessária conexão com a internet para utilizar o aplicativo.

Requisitos mínimos:

  • Sistema operacional: Windows 7 ou superior
  • Navegador: Chrome, Firefox, Edge ou Safari atualizado
  • Leitor de PDF instalado ou navegador com suporte a visualização de PDF

Solução de problemas:

  • Se o aplicativo não abrir automaticamente, navegue até a pasta do pendrive e dê um duplo clique no arquivo index.html
  • Caso os PDFs não abram, verifique se seu navegador tem suporte à visualização de PDFs ou instale um leitor como Adobe Reader
  • Se as imagens não aparecerem, certifique-se de que a pasta images está presente no pendrive
